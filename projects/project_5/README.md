## Описание проекта

**Цель проекта:** Построение модели машинного обучения для точного предсказания длительности поездок такси в Нью-Йорке.

**Бизнес-задача:** Оптимизация работы таксопарков через точное прогнозирование времени поездок, улучшение планирования и распределения ресурсов.

**Техническая задача:** Решение задачи регрессии с использованием различных алгоритмов машинного обучения и глубоким feature engineering.

**Ключевые этапы:**
1. EDA и анализ распределений
2. Feature engineering: временные, географические, погодные признаки
3. Создание полиномиальных признаков и кластеризация
4. Построение и сравнение моделей регрессии
5. Оптимизация гиперпараметров
6. Прогнозирование на тестовых данных

## Описание данных

Проект использует датасет [NYC Taxi Trip Duration](https://www.kaggle.com/competitions/nyc-taxi-trip-duration/overview) от Kaggle, обогащенный дополнительными данными.

**Основные источники данных:**
- **Основные данные:** 1.5M+ записей о поездках такси NYC (январь-июнь 2016)
- **OSRM данные:** Расчетные расстояния и время по маршрутам
- **Погодные данные:** Температура, осадки, влажность
- **Геоданные:** Координаты начала/окончания поездок

**Ключевые признаки:**
- Временные: дата/время, день недели, час, праздники
- Географические: координаты, хаверсинус-расстояние, 10 географических кластеров
- Маршрутные: расчетное расстояние, количество шагов маршрута
- Погодные: температура, осадки
- Бизнес-факторы: таксопарк, количество пассажиров, флаг хранения данных

**Целевая переменная:** Длительность поездки в секундах (логарифмированная для обучения)

## Результаты

**Лучшая модель:** Gradient Boosting (XGBoost)  
**RMSLE на валидации:** 0.39  
**Медианная абсолютная ошибка:** 1.8 минуты  
**Объем данных:** 1.5+ млн поездок

**Сравнение моделей:**
- Gradient Boosting: RMSLE = 0.39
- Random Forest: RMSLE = 0.41
- Decision Tree: RMSLE = 0.41
- Linear Regression: RMSLE = 0.47

**Ключевые фичи:**
- Временные признаки (час, день недели, праздники)
- Географические кластеры (10 кластеров)
- OSRM данные (расстояние, время маршрута)
- Погодные условия
