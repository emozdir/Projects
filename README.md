## Описание проекта

**Цель проекта:** Построение модели машинного обучения для прогнозирования вероятности открытия депозита клиентом банка на основе данных маркетинговой кампании.

**Бизнес-задача:** Повысить эффективность прямого маркетинга за счет таргетирования клиентов с наибольшей вероятностью отклика.

**Техническая задача:** Решение задачи бинарной классификации с использованием различных алгоритмов машинного обучения.

**Структура проекта:**
- `Project_4_ML.ipynb` - основной ноутбук с полным анализом и моделированием
- `data/bank.csv` - исходные данные маркетинговой кампании
- `requirements.txt` - файл с зависимостями

## Описание данных

Данные содержат информацию о клиентах банка и результатах маркетинговой кампании по привлечению депозитных вкладов.

**Основные признаки:**
- Демографические данные: возраст, профессия, семейное положение, образование
- Финансовые показатели: баланс, наличие кредитов
- Данные кампании: тип контакта, месяц, длительность звонка
- Экономические индикаторы: индекс потребительских цен, уровень безработицы
- Целевая переменная: факт открытия депозита (yes/no)

**Источник данных:** [UCI Bank Marketing Dataset](https://archive.ics.uci.edu/ml/datasets/Bank+Marketing)
